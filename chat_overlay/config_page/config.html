User Total Messages Threshold

Indicates to the bot to show chat messages from any user if the user has less than 'x' total messages so far today in the channel, where 'x' is the value input here.

For example: if you set 2, the first two messages any user enters into your twitch channel is guaranteed be shown by the overlay chat app

User Total Bits Donated Threshold

Indicates to the bot to show messages from any user if the user has donated more than 'x' number of bits, where 'x' is the value input here.

For example: if you set 5000, any messages from users who have donated more than 5000 bits are guaranteed to be shown in the overlay chat app window


About

BearlyChat is a twitch chat application used to customize a set of filters and display a translucent overlay with the filtered chat so you never miss an important message in your chat.

Whether its a new user joining your channel for the first time, a valued subscriber/donator, or someone asking a question to you directly you'll see it instantly without all the other chatter!
